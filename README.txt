# Course Registration System

This project is a prototype web application designed to simplify the course registration process for students. It allows users to search, filter, and select courses while managing a schedule and waitlist.


# Live Application
https://chessnoec-bot.github.io/online-course-registration-ui/

# GitHub Repository
https://github.com/chessnoec-bot/online-course-registration-ui

# Features
- Search courses by name, code, or professor
- Filter courses by availability (open/full)
- Sort courses by code, name, or availability
- Add courses to schedule
- Join waitlist for full courses
- View detailed course information
- Real-time message feedback system

# Technologies Used
- HTML
- CSS
- JavaScript

# Database
 We used an JavaScript array to simulate a data within the database.

# Server-Side
This project does not use server-side components.  
All functionality is handled on the frontend.

# How to Run
1. Open the live application link above OR  
2. Download the project files  
3. Open `index.html` in any web browser  

# User Guide
- Browse available courses in the Course Selection panel
- Use the search bar to find courses by code, name, or professor
- Use filters to display open or full courses
- Sort courses based on your preference (code, name, availability)
- Click **"Info"** to view course details
- Click **"Add"** to add a course to your schedule
- Click **"Join Waitlist"** if a course is full
- View selected courses in the **My Schedule** panel
- View waitlisted courses in the **My Waitlist** panel
- Remove courses at any time using the **Remove** button
- System messages will appear in the **Student Message Center**
